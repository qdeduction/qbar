Installation

Warning!

Right now, qbar is a work in progress, in the pre-α stage, so the API will fluctuate wildly. Operate at your own risk!

Cargo

To install with cargo, run the following in a command prompt:

cargo install qbar

To add the qbar API library to a Rust project, add the following to your Cargo.toml:

[dependencies]
qbar = 0.0.0

Other Installation Methods

There are no other installation methods at the moment, but you can download or clone the source code in the meantime.
